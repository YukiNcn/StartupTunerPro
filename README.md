# StartupTunerPro
An automatic phone optimization Magisk module for boot time
—————————————————————————————————————
HISTORY
This Magisk module is adapted from a post by Coolapk user @Yz堕仙: https://www.coolapk.com/feed/65005997?s=YjdmY2UzYzAyM2ExNDg4ZzY5NjBmZDRjega1591
—————————————————————————————————————
FEATURE
Original Features:
1.Automatically fixes SDK interface anomalies after first unlock post-boot
2.Disables USB debugging
3.Deletes TWRP folder
4.Enables SELinux enforcing mode
5.Turns off Developer options
6.Auto-starts BlackDomain (黑阈)
7.Uses random serial number generated before shutdown (not new)

New Features:
1.Automatically generates a new random serial number after first unlock post-boot
2.Auto-starts HxAudio
————————————————————————————————————
NOTE
Random serial number generation, auto-start BlackDomain (黑阈), and auto-start HxAudio can be enabled or disabled via volume key selection when flashing the module.
